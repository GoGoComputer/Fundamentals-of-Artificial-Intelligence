# 7장. 더 넓은 AI 세계

> 1~6장에서 다룬 건 머신러닝의 **지도학습 (분류 + 회귀)** 과 그것을 운영하는 법이었습니다.
> 그게 ML의 가장 큰 부분이지만, 전부는 아닙니다.
>
> 이 장은 **"AI에는 또 무엇이 있는가"** 의 큰 그림을 그려 드립니다.
> 깊이는 입문 수준이고, 각 주제마다 더 가고 싶으시면 다음 학습 자료를 안내드려요.

---

---

## 이 장의 위치

지금까지 배우신 1~6장은 ML 엔지니어의 **공통 기본기** 였습니다. 이 장은 본인의 관심사에 따라 **선택적으로** 보시면 돼요.

```
관심사            → 이 장에서 추천
─────────────────────────────────────
이미지·영상       → 7.2 (CNN)
자연어·번역       → 7.4 (Transformer)
시계열·예측       → 7.3 (RNN/LSTM), 7.4
추천·검색         → 7.1 (비지도학습)
강화학습 / 게임    → 7.7
실무 안정성       → 7.5 (EDA), 7.6 (해석), 7.8 (윤리)
```

처음부터 끝까지 순서대로 보지 않으셔도 됩니다. 본인 분야에 맞는 글부터 펴 보세요.

---

## 이 장이 다루지 않는 것

**완벽한 깊이**는 다루지 않습니다. 각 주제마다 책 한 권씩 분량이 따로 있어요. 이 장은 "이게 뭔지, 어디에 쓰이는지, 다음에 무엇을 봐야 하는지" 까지만 안내합니다.

각 글의 끝에 **"더 깊이 가고 싶으시면"** 섹션이 있어요. 본격적으로 한 분야를 파고드실 땐 그 자료들을 보시면 됩니다.

자, 시작해 보시죠.
