## 1.1 변수와 식

수학에서 **변수(variable)** 는 "아직 모르는 숫자"를 글자로 적어둔 거예요.

```
x = 5
y = 3 × x + 2 = 17
```

`x`라는 글자 안에 숫자 5가 들어 있다고 생각하시면 됩니다.

### "왜 글자를 써요?"

같은 식을 여러 값에 적용할 수 있어서요. `x`가 5일 때, 10일 때, 100일 때 — 같은 식을 그대로 써요.

```
x = 5  → y = 3 × 5 + 2 = 17
x = 10 → y = 3 × 10 + 2 = 32
x = 100 → y = 3 × 100 + 2 = 302
```

### ML에서?

머신러닝에선 항상 변수 이야기예요.
- `x` = 입력 (사진의 픽셀, 고객의 정보 등)
- `y` = 출력 (예측 결과)
- `w` = 가중치 (모델이 학습하는 값)

```
y = w × x + b
```

이게 사실 가장 단순한 ML 모델이에요. 다음에서 자세히.

## 1.2 함수 (Function)

**함수**는 **"입력을 받으면 정해진 규칙으로 출력을 내놓는 기계"** 예요.

```
입력: x → [함수 f] → 출력: f(x)

예: f(x) = 2x + 3

f(0) = 3      (x=0을 넣으면 3이 나옴)
f(1) = 5
f(2) = 7
f(10) = 23
```

자판기 같은 거예요. 동전(입력)을 넣으면 정해진 음료수(출력)가 나오는.

### 표기법

`f(x)` 라고 쓰면 "x를 입력으로 받는 함수 f". `f` 대신 `g`, `h` 등도 자주 써요.

```
f(x) = 2x + 3       ← x → 2x + 3
g(x) = x²            ← x → x²
h(x) = √x            ← x → √x (제곱근)
```

### 함수의 값 구하기

`f(x) = 2x + 3` 이라고 할 때:
- `f(5) = ?` → x 자리에 5를 넣어요 → `f(5) = 2 × 5 + 3 = 13`

이걸 **"함수에 5를 대입한다"** 라고 해요.

### 함수가 ML의 핵심

다시 말하지만, **머신러닝 모델은 함수예요.**

```
입력 (이미지 픽셀들) → [모델 f] → 출력 (강아지/고양이)

이미지 → f(이미지) → 0.92 (강아지일 확률)
```

`model(x)`라고 코드에서 쓸 때마다, 사실은 함수의 값을 구하고 있는 거예요.

## 1.3 좌표평면 (Coordinate Plane)

함수를 그림으로 그리려면 **좌표평면**이 필요해요. 가로축(x축)과 세로축(y축)이 있는 격자.

```
       y
       ↑
    5  ┤
    4  ┤        ●  (3, 4)
    3  ┤
    2  ┤  ●     
    1  ┤  (1, 2)
    0  ┼──┼──┼──┼──┼──→ x
       0  1  2  3  4  5
```

`(x, y)` 형태로 점을 찍어요. 처음 숫자는 가로 위치, 두 번째 숫자는 세로 위치.

- `(1, 2)`: 가로 1, 세로 2
- `(3, 4)`: 가로 3, 세로 4

### 함수의 그래프

`y = f(x)` 라는 함수를 그래프로 그리려면, **여러 x 값에 대해 y 값을 구해서 점으로 찍고 이어 주면** 돼요.

`f(x) = x` 의 그래프:

```
x: 0, 1, 2, 3, 4
y: 0, 1, 2, 3, 4

       y
       ↑
    4  ┤            ●
    3  ┤         ●
    2  ┤      ●
    1  ┤   ●
    0  ┼●─┼──┼──┼──┼──→ x
       0  1  2  3  4
```

이 점들을 이으면 직선이 되죠. 이게 **`y = x`의 그래프**예요.

## 1.4 1차 함수 (선형 함수)

`y = ax + b` 형태의 함수예요. 그래프가 항상 **직선**이에요.

### 핵심: 기울기와 절편

```
y = 2x + 3
    ↑   ↑
   기울기  y절편
```

- **기울기 (a)**: x가 1 늘 때 y가 얼마나 늘어나는지. 위 식에서는 2.
- **y절편 (b)**: x가 0일 때 y의 값. 위 식에서는 3.

### 그래프 그려보기

`y = 2x + 3`:

```
x: -2, -1, 0, 1, 2
y: -1,  1, 3, 5, 7

       y
       ↑
    7  ┤              ●
    6  ┤            ╱
    5  ┤         ●
    4  ┤        ╱
    3  ┤      ●  (y절편)
    2  ┤    ╱
    1  ┤  ●
    0  ┼──┼──┼──┼──┼──→ x
       0  1  2  3  4
```

직선이죠. 1차 함수는 항상 이래요.

### 기울기의 의미

기울기가 크면 직선이 가팔라요. 작으면 완만해요.

```
y = 0.5x  (기울기 0.5, 완만)
       y
    2  ┤     ╱
    1  ┤   ╱
    0  ┼─┼─┼─→ x
       0  1  2

y = 2x  (기울기 2, 가파름)
       y
    4  ┤  ╱
    3  ┤ ╱
    2  ┤╱
    1  ┤
    0  ┼─┼──→ x
       0  1  2
```

기울기가 음수면 오른쪽으로 갈수록 내려가요.

```
y = -x  (기울기 -1)
       y
    1  ●
    0  ┼●─┼──→ x
   -1     ●
   -2        ●
```

### ML에서?

선형 회귀 모델이 **정확히 1차 함수**예요.

```
집값 = (기울기) × 평수 + (y절편)

예: 집값 = 0.5 × 평수 + 1
   30평 → 0.5×30 + 1 = 16억
```

머신러닝이 하는 일이 바로 **이 기울기와 y절편을 데이터로부터 알아내는 것**이에요. 본문 [3.3 선형회귀](../../03-머신러닝-회귀/이론/03-선형회귀.md)에서 봤죠.

## 1.5 2차 함수 (포물선)

`y = ax² + bx + c` 형태. 그래프가 **포물선**이에요.

### 가장 단순한 2차 함수: y = x²

```
x: -3, -2, -1, 0, 1, 2, 3
y:  9,  4,  1, 0, 1, 4, 9

       y
       ↑
    9  ┤●           ●
    8  ┤
    7  ┤ \         /
    6  ┤
    5  ┤  \       /
    4  ┤   ●     ●
    3  ┤
    2  ┤    \   /
    1  ┤     ● ●
    0  ┼──────●─────→ x
      -3 -2 -1 0 1 2 3
```

U자 모양이에요. 이걸 **포물선**이라고 해요.

### 핵심 특성

1. **최솟값(또는 최댓값) 이 있음.** `y = x²`은 x=0에서 y=0이 최소.
2. **양쪽으로 갈수록 빠르게 증가.** x²이라서 x가 두 배 되면 y는 4배.

### ML에서 왜 중요?

가장 자주 만나는 손실 함수, **MSE**가 사실 2차 함수예요.

```
손실 L(w) = (예측 - 실제)² = (w·x - y)²
         ↑
       w에 대한 2차 함수
```

손실 함수가 2차 함수면, 이런 모양이에요.

```
L
│
│ \                /
│  \              /
│   \            /     ← w가 어디 있을 때
│    \          /         손실이 최소가 될까?
│     \________/
│         ↑
│      여기!
└─────────────→ w
```

**가운데 가장 낮은 점**을 찾는 게 학습이에요. 그래서 미분이 필요한 거고요. ([3장](03-미분-입문.md)에서 자세히)

## 1.6 방정식과 부등식

### 방정식

`x` 값을 모를 때, **"이런 조건이면 x는 무엇이어야 하지?"** 를 묻는 식이에요.

```
2x + 3 = 11
       (해 = ?)

풀이:
2x + 3 = 11
2x = 11 - 3
2x = 8
x = 4
```

해는 `x = 4`. 이걸 식에 다시 넣어보면 `2×4 + 3 = 11` ✓.

### 부등식

`=` 대신 `<`, `>`, `≤`, `≥` 가 들어간 거예요.

```
2x + 3 < 11
2x < 8
x < 4
```

`x`는 4보다 작은 모든 수.

### ML에서 왜?

학습 중에 자주 만나요.

- "정확도가 90%보다 크면 학습 멈춰" → `accuracy > 0.9`
- "loss가 0.01보다 작아질 때까지 반복" → `loss < 0.01`

조건문에 무조건 들어가는 형태예요.

## 1.7 피타고라스 정리

직각삼각형에서 **빗변의 제곱 = 두 변의 제곱의 합.**

```
        |\
        | \
        |  \  c
      a |   \
        |    \
        |_____\
           b

a² + b² = c²
```

예: a=3, b=4 → c² = 9 + 16 = 25 → c = 5

### ML에서 왜?

두 점 사이의 거리를 계산할 때 써요.

```
점 A = (1, 2)
점 B = (4, 6)

A에서 B까지 거리?
가로 차이: 4-1 = 3
세로 차이: 6-2 = 4
거리 = √(3² + 4²) = √25 = 5
```

KNN 같은 알고리즘이 정확히 이 거리를 계산해요. 모든 점 사이 거리를 재서 가까운 K개를 찾는 거죠.

3차원 이상에서도 똑같아요.

```
3D에서 거리 = √(Δx² + Δy² + Δz²)
n차원에서 거리 = √(Δx₁² + Δx₂² + ... + Δxₙ²)
```

이걸 **유클리드 거리(Euclidean distance)** 라고 해요. ML에서 정말 자주 만나요.

## 1.8 % (백분율)

100분의 몇이라는 뜻. `30%` = `30/100` = `0.3`.

### ML에서?

- 정확도: `accuracy = 0.92` = 92%
- 학습 데이터 비율: `test_size=0.2` = 20%
- Dropout 비율: `p=0.3` = 30%

본문 곳곳에 나와요.

## 1.9 정리: 이 단원에서 챙길 것

```python
# 변수와 식
y = 2 * x + 3       # x를 받아 y를 계산

# 함수
def f(x):
    return 2 * x + 3

# 좌표평면 — (x, y) 점들
# 1차 함수 — 직선, 기울기와 절편
# 2차 함수 — 포물선, 최솟값
# 방정식 — = 모르는 값 찾기
# 부등식 — < > 로 조건 표현
# 거리 — √(차이²의 합)
```

### ML과의 연결

| 중학 수학 | ML에서 어디? |
|----------|-------------|
| 함수 | 모델 자체 (`model(x) = y`) |
| 1차 함수 | 선형 회귀 |
| 2차 함수 | MSE 손실 (가중치에 대한) |
| 좌표평면 | 데이터 시각화 (산점도) |
| 거리 (피타고라스) | KNN, K-Means 등 |

다 익숙해지셨다면, [고등 수학 핵심](02-고등수학-핵심.md)으로 가 봅시다.
