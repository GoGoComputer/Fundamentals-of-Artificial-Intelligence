# 1.3 연산자: 더하기부터 비교까지

## 연산자가 뭐예요?

연산자는 **값과 값 사이에 끼워서 무언가를 계산하게 하는 기호**입니다. 사실 우리가 초등학교 때 다 배운 거예요.

```python
2 + 3    # 5
10 - 4   # 6
```

`+`, `-` 이런 게 다 연산자예요. 파이썬에는 크게 4가지 종류가 있습니다.

1. **산술 연산자** — 더하기, 빼기, 곱하기 등
2. **비교 연산자** — 같다, 크다, 작다
3. **논리 연산자** — and, or, not
4. **할당 연산자** — `=`, `+=` 같은 것들

이번엔 멤버십 연산자(`in`)도 잠깐 다룰게요.

---

## 1. 산술 연산자

### 기본 4종

```python
print(10 + 3)    # 13   (덧셈)
print(10 - 3)    # 7    (뺄셈)
print(10 * 3)    # 30   (곱셈)
print(10 / 3)    # 3.333... (나눗셈)
```

⚠️ **주의:** 나눗셈 `/`은 항상 **실수(float)** 를 반환합니다. `10 / 2`도 `5.0`이지 `5`가 아니에요.

### 좀 특이한 3종

```python
print(10 // 3)   # 3    (몫만)
print(10 % 3)    # 1    (나머지만)
print(10 ** 3)   # 1000 (10의 3제곱)
```

| 연산자 | 이름 | 예시 | 결과 |
|------|------|------|------|
| `//` | 몫 (정수 나눗셈) | `7 // 2` | `3` |
| `%` | 나머지 | `7 % 2` | `1` |
| `**` | 거듭제곱 | `2 ** 10` | `1024` |

### 어디에 써요?

**`%` (나머지)** 는 의외로 정말 자주 씁니다. "짝수냐 홀수냐"를 확인할 때나, "5의 배수마다 뭔가 하기" 같은 일에요.

```python
n = 14

if n % 2 == 0:
    print("짝수")
else:
    print("홀수")
```

**`**`** 는 거듭제곱이지만, 머신러닝에서는 "제곱근"을 위해 자주 씁니다.

```python
print(16 ** 0.5)    # 4.0   (16의 1/2제곱 = 제곱근)
print(8 ** (1/3))   # 2.0   (8의 1/3제곱 = 세제곱근)
```

### 연산 순서 (우선순위)

학교에서 배운 그대로예요. 곱셈/나눗셈이 덧셈/뺄셈보다 먼저, 거듭제곱이 가장 먼저.

```python
print(2 + 3 * 4)        # 14   (3*4=12 먼저)
print((2 + 3) * 4)      # 20   (괄호 먼저)
print(2 ** 3 * 4)       # 32   (2**3=8 먼저)
```

**기억 안 나면 그냥 괄호 쓰세요.** 코드 가독성도 좋아져요. 우선순위를 다 외우려고 애쓰지 마세요.

```python
# 좀 더 명확하게
result = (2 ** 3) * 4
```

## 2. 비교 연산자

값을 비교하면 결과는 **`True` 또는 `False`** 로 나옵니다.

```python
print(5 > 3)     # True
print(5 < 3)     # False
print(5 == 5)    # True
print(5 != 5)    # False
print(5 >= 5)    # True
print(5 <= 4)    # False
```

| 연산자 | 의미 |
|------|------|
| `==` | 같다 |
| `!=` | 다르다 |
| `>` | 크다 |
| `<` | 작다 |
| `>=` | 크거나 같다 |
| `<=` | 작거나 같다 |

⚠️ **주의:** **"같다"는 `=`이 아니라 `==` 입니다!**

`=`은 "왼쪽에 오른쪽을 담아라"라는 할당이에요. 비교는 `==`을 두 개 써야 해요. 처음에 정말 자주 헷갈리는 부분입니다.

```python
age = 25       # 할당: age라는 이름표를 25에 붙임
age == 25      # 비교: age가 25랑 같은지? → True
```

### 문자열도 비교됩니다

```python
print("apple" == "apple")    # True
print("apple" == "Apple")    # False  (대소문자 구분!)
print("apple" < "banana")    # True   (사전 순서)
```

문자열을 비교하면 **사전 순서**로 비교돼요. `apple`이 `banana`보다 사전에서 먼저 나오니까 작다고 판단합니다.

### 연쇄 비교

이건 파이썬의 예쁜 기능 중 하나예요.

```python
age = 25
if 20 <= age < 30:
    print("20대")
```

`20 <= age < 30`이 `(20 <= age) and (age < 30)`을 한 줄에 쓴 것과 같아요. 다른 언어에서는 안 되지만 파이썬에서는 됩니다.

## 3. 논리 연산자

`True/False` 값들끼리 결합할 때 씁니다.

| 연산자 | 의미 | 예시 |
|------|------|------|
| `and` | 둘 다 참이면 참 | `True and True` → `True` |
| `or` | 하나라도 참이면 참 | `True or False` → `True` |
| `not` | 반대로 뒤집기 | `not True` → `False` |

```python
age = 25
has_license = True

# 운전할 수 있나?
can_drive = (age >= 18) and has_license
print(can_drive)    # True

# 학생 또는 노인이면 할인
is_student = False
is_senior = True
can_discount = is_student or is_senior
print(can_discount)    # True

# 미성년자가 아니면
is_minor = age < 18
can_vote = not is_minor
print(can_vote)    # True
```

### 진리표 (외울 필요 없어요)

| A | B | A and B | A or B |
|---|---|---------|--------|
| True | True | True | True |
| True | False | False | True |
| False | True | False | True |
| False | False | False | False |

**한 줄 요약:**
- `and`: 둘 다 참이어야 참
- `or`: 하나만 참이어도 참
- `not`: 뒤집기

## 4. 할당 연산자

이미 알고 계신 `=`이 가장 기본 할당 연산자예요. 그런데 자주 쓰는 단축형들이 있어요.

```python
count = 0

count = count + 1    # count를 1 증가
count += 1           # 위와 똑같음 (단축형)
```

| 연산자 | 의미 |
|------|------|
| `+=` | 더한 후 할당 |
| `-=` | 뺀 후 할당 |
| `*=` | 곱한 후 할당 |
| `/=` | 나눈 후 할당 |
| `**=` | 거듭제곱 후 할당 |

```python
score = 100
score += 10     # score = score + 10  → 110
score -= 5      # 105
score *= 2      # 210
score /= 3      # 70.0
print(score)
```

머신러닝 코드에서 **누적 합** 계산할 때 자주 봅니다.

```python
total_loss = 0
for batch in batches:
    loss = compute_loss(batch)
    total_loss += loss   # ← 이런 식
```

## 5. 멤버십 연산자: in, not in

이건 **"안에 있는지 확인"** 하는 연산자예요. 다음 글의 자료구조에서 본격적으로 쓰지만, 지금 미리 봐도 좋아요.

```python
fruits = ["사과", "바나나", "딸기"]

print("사과" in fruits)        # True
print("포도" in fruits)        # False
print("포도" not in fruits)    # True
```

문자열 안에 특정 글자가 있는지도 확인할 수 있어요.

```python
sentence = "오늘 날씨가 좋다"
print("날씨" in sentence)    # True
print("비" in sentence)      # False
```

## 모두 합쳐 보기

이제 진짜 코드 같은 걸 짜 봅시다. 입장 가능 여부를 확인하는 코드예요.

```python
age = 25
ticket = True
is_member = False
points = 1500

# 입장 조건:
# 1. 나이가 19세 이상
# 2. 티켓이 있거나, 회원이거나, 포인트가 1000 이상
can_enter = (age >= 19) and (ticket or is_member or points >= 1000)

print(f"입장 가능: {can_enter}")    # 입장 가능: True
```

이 한 줄에 비교 연산자(`>=`), 논리 연산자(`and`, `or`), f-string이 다 들어 있어요.

## 정리: 연산자 한 페이지 요약

```python
# 산술
+   -   *   /     # 덧셈, 뺄셈, 곱셈, 나눗셈
//  %   **        # 몫, 나머지, 거듭제곱

# 비교 (결과는 True/False)
==  !=            # 같다, 다르다
>   <   >=  <=    # 크다, 작다, 이상, 이하

# 논리
and   or   not    # 둘 다, 하나라도, 뒤집기

# 할당 단축
+=  -=  *=  /=    # x = x + 1 → x += 1

# 멤버십
in    not in      # 안에 있냐, 없냐
```

## 자주 묻는 질문

> **Q. `=` 와 `==` 정말 매번 헷갈려요. 팁 있나요?**
>
> 이렇게 외우시면 도움 됩니다.
> - `=` 한 개 → "값을 넣어라" (할당)
> - `==` 두 개 → "값이 같냐?" (비교)
>
> 그리고 IDE/에디터들이 `if x = 5:` 같이 쓰면 빨간 줄로 알려줘요. 그것도 도움이 됩니다.

> **Q. `and`와 `&&` 차이는 뭐예요?**
>
> 자바스크립트나 C 같은 언어에서는 `&&`을 쓰지만, **파이썬은 `and`를 씁니다.** 영어 단어 그대로요. 처음에는 어색해도 금방 익숙해져요.

> **Q. `5 + 3 * 2`는 11이지만 `(5 + 3) * 2`는 16인 거 맞죠?**
>
> 맞습니다. 곱셈이 덧셈보다 먼저예요. 모든 우선순위를 외우려 하지 말고, **헷갈리면 괄호를 쓰세요.** 코드 읽기에도 더 좋습니다.

> **Q. `is`와 `==`은 뭐가 다른가요?**
>
> 좋은 질문이지만 지금 단계에서는 그냥 **`==`만 쓰세요.** `is`는 "같은 메모리 주소냐?"를 묻는 거고, `==`는 "값이 같냐?"를 묻습니다. 거의 항상 후자가 우리가 원하는 거예요. 단, **`x is None` 같은 경우는 예외**로 `is`를 씁니다 (이건 관례).
