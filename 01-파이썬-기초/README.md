## 공부하는 법

### 1. 글을 먼저 읽으세요
이론 파일을 한 번 죽 읽어 주세요. 모르는 단어가 있어도 일단 끝까지 읽으세요.

### 2. 코드를 손으로 따라 치세요
**복사 붙여넣기 금지입니다.** 한 글자씩 직접 치세요. 손가락이 기억하는 게 가장 오래 갑니다.

### 3. 결과를 예측하고 실행하세요
"이 코드는 이런 결과가 나올 거 같아" 하고 예상해 보세요. 그다음 실행해 보세요. 예상과 결과가 다르면, **그 순간이 진짜 배우는 순간**입니다.

### 4. 의도적으로 망가뜨려 보세요
잘 돌아가는 코드를 일부러 한 글자씩 바꿔 보세요. `print` 대신 `prnt`로, `:` 빼 보고, 숫자에 따옴표 씌워 보고. 어떤 에러가 나는지 보세요. 에러를 친구로 만들어야 합니다.

## 이미 파이썬 아시는 분에게

이 장은 파이썬 입문서가 아닙니다. **머신러닝에 필요한 파이썬만 골라 담은 압축본**이에요. 이미 파이썬을 어느 정도 쓰실 수 있으시다면, 다음만 빠르게 확인하시고 [2장](../02-머신러닝-분류/)으로 넘어가셔도 됩니다.

- ✅ 리스트와 딕셔너리를 자유롭게 다룰 수 있다
- ✅ for문과 if문을 중첩해서 쓸 수 있다
- ✅ 함수를 정의하고 호출할 수 있다
- ✅ `import numpy as np` 같은 import 문법을 안다

이 네 가지가 다 OK면 1장은 건너뛰셔도 됩니다.

## 일부러 안 다루는 것들

다음 주제들은 파이썬에서 중요하지만, **머신러닝 입문 단계에서는 굳이 알 필요가 없어서** 빼두었습니다.

- 클래스와 객체지향 (4장에서 PyTorch 모델 만들 때 필요한 만큼만 다룹니다)
- 예외 처리 (try/except)
- 제너레이터, 데코레이터, 메타클래스
- 비동기 (async/await)
- 컨텍스트 매니저 (with문)
- 정규표현식

이 자료를 다 보시고 더 깊이 들어가고 싶으시다면 [부록의 더 나아가기](../부록/더-나아가기.md)를 참고해 주세요.
