## 시작하기 전에

### GPU를 켜세요

이 장부터는 **GPU가 필요**합니다. Colab에서:

1. 메뉴에서 **`런타임 → 런타임 유형 변경`**
2. **하드웨어 가속기를 "GPU"** 로 변경
3. 저장

확인하기:
```python
import torch
print(torch.cuda.is_available())   # True여야 함
```

CPU로도 돌릴 수 있지만 학습이 매우 느려요. GPU 강력 추천.

### 우리는 PyTorch를 씁니다

딥러닝 프레임워크는 크게 두 가지가 있어요.

| | TensorFlow / Keras | **PyTorch** |
|---|------|------|
| 만든 곳 | Google | Meta (Facebook) |
| 인기 (2026) | 안정 | **상승 중** (학계 95%, 산업 50%+) |
| 문법 | 다소 복잡 | **Pythonic, 직관적** |
| 디버깅 | 어려움 | 쉬움 |

이 자료는 **PyTorch**를 씁니다. 이유:
1. 학계의 거의 모든 새 모델이 PyTorch
2. 문법이 더 직관적
3. 회사들도 점점 PyTorch로 옮겨가는 추세

TensorFlow를 배우셔야 한다면, PyTorch를 익히신 다음 며칠이면 옮기실 수 있어요.
