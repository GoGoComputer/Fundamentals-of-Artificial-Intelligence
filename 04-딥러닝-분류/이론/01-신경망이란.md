# 4.1 신경망이란 무엇인가

## 사실은 별 거 아니에요

"신경망(Neural Network)" 이라는 단어를 듣고 뇌를 떠올리셨다면, 일단 잊어 주세요. 그건 마케팅 비유고요. **수학적으로 신경망은 다음 두 가지를 반복하는 작은 함수**예요.

> 1. **곱하고 더하기** (선형 결합)
> 2. **꾸불꾸불하게 만들기** (활성화 함수)

이게 전부입니다. 진짜로요. 챗GPT, 알파고, 자율주행 — 다 이 두 가지의 반복이에요.

---

## 가장 간단한 신경망: 직선 하나

먼저 우리가 이미 아는 걸 떠올려 봅시다. 선형 회귀.

```
y = w₁x₁ + w₂x₂ + ... + b
```

이 식이 사실은 **가장 단순한 신경망**이에요. 입력들을 곱하고 더한 거죠.

```
       x₁
        \  w₁
         \
   x₂────[+]────y
        /  w₂
       /
       (+ b)
```

이걸 **퍼셉트론(perceptron)** 이라고도 불러요. 1958년에 만들어졌어요.

---

## 한 단계 위: 활성화 함수 추가

선형 회귀에는 한 가지 한계가 있어요. **직선밖에 못 그려요.** 곡선이 필요한 데이터엔 약하죠.

그래서 결과에 한 번 더 처리를 해 줍니다. **꾸불꾸불하게 만드는 함수**를 통과시키는 거예요.

```
y = activation(w₁x₁ + w₂x₂ + ... + b)
                ───────────────────────
                       선형 결합
```

이 `activation`이 **활성화 함수(activation function)** 입니다. 가장 자주 쓰는 게 **ReLU**예요.

```
ReLU(x) = max(0, x)

  y │
    │      /
    │    /
    │  /
   ─┼─────────── x
    │/
    │
```

음수는 0으로, 양수는 그대로. 이렇게 단순해요.

---

## 진짜 신경망: 층(layer)을 쌓기

지금까지는 **한 줄짜리** 신경망이었어요. 이걸 **여러 층**으로 쌓으면 진짜 신경망이 됩니다.

```
입력      층 1            층 2           층 3        출력
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

x₁ ─┐                ┌──[+]──ReLU──┐                
     ├──[+]──ReLU──┤              ├──[+]──┐         
x₂ ─┘                └──[+]──ReLU──┘       │        
                                            ├─[+]─→ y
x₃ ─┐                ┌──[+]──ReLU──┐       │        
     ├──[+]──ReLU──┤              ├──[+]──┘         
x₄ ─┘                └──[+]──ReLU──┘
```

각 층은 **이전 층의 출력을 입력으로 받아서, 다시 곱하고 더하고 활성화**합니다.

층이 많아질수록(=깊어질수록) 표현할 수 있는 함수가 복잡해져요. 그래서 **deep** learning이라고 부릅니다.

---

## "왜 깊으면 더 강력한가요?"

좋은 질문입니다. 직관적으로 말하면, **층마다 점점 더 추상적인 패턴을 학습**합니다.

이미지로 강아지 분류를 한다고 해 봐요.

```
입력 (픽셀)
    ↓
층 1: 선과 모서리 (vertical, horizontal lines)
    ↓
층 2: 모양 (눈, 코, 귀의 부분)
    ↓
층 3: 부품 (얼굴, 다리, 꼬리)
    ↓
층 4: 객체 (강아지!)
    ↓
출력 (확률)
```

층이 깊어질수록 **점점 더 큰 그림**을 보는 거예요. 이게 딥러닝이 강력한 이유 중 하나입니다.

---

## 분류용 신경망: MNIST 예시

MNIST 손글씨를 분류하는 신경망을 그려 봅시다.

```
입력: 784 픽셀 (28x28 이미지를 펴서)
   ↓
층 1: 256 노드 (Dense + ReLU)
   ↓
층 2: 128 노드 (Dense + ReLU)
   ↓
출력: 10 노드 (각 숫자에 대한 점수)
```

각 노드가 **이전 층의 모든 노드와 연결**되어 있어요. 이런 층을 **Fully Connected Layer** 또는 **Dense Layer**라고 부릅니다.

### 출력층 해석

마지막 10개의 노드는 각각 **0~9 숫자에 대한 모델의 점수**예요. 가장 큰 값을 가진 인덱스가 모델의 답입니다.

```python
output = model(image)
# output: [-0.3, 1.2, 0.8, 5.7, 0.1, ...] (10개 숫자)

prediction = output.argmax()    # 가장 큰 값의 인덱스
# prediction: 3  (3번이 가장 크면)
```

**확률**로 보고 싶으면 `Softmax` 함수를 쓰면 돼요.

```python
import torch.nn.functional as F
probs = F.softmax(output, dim=-1)
# probs: [0.001, 0.05, 0.02, 0.85, 0.005, ...]   (합쳐서 1)
```

## 학습이란?

이제 학습이 뭔지 봐야 해요. 학습이라는 건 **가중치들을 좋은 값으로 찾아가는 과정**이에요.

```
처음: 가중치 = 무작위 → 모델 답이 엉터리
   ↓ 학습
나중: 가중치 = 학습된 값 → 모델 답이 정확
```

이 학습이 어떻게 일어나는지는 다음 세 가지 글에서 자세히 다룹니다.

1. [PyTorch 시작하기](02-PyTorch-시작.md)
2. [첫 신경망 만들기](03-첫-신경망.md)
3. [학습 과정 들여다보기](04-학습-과정.md)

지금은 큰 그림만:

```
1. 신경망에 데이터를 통과시킨다 (forward)
2. 답이 얼마나 틀렸는지 점수를 매긴다 (loss)
3. 그 점수를 줄이는 방향으로 가중치를 살짝 바꾼다 (backward + step)
4. 1~3을 수만~수백만 번 반복한다 (epochs)
```

이게 **모든 신경망 학습의 본질**이에요. ChatGPT도 이렇게 학습됐습니다 (단지 데이터가 인터넷 전체였을 뿐).

---

## 신경망 vs 머신러닝

지금까지 배운 ML 모델들과 신경망을 비교해 봅시다.

| | ML (Random Forest 등) | 신경망 (DL) |
|---|---|---|
| **데이터 양** | 1천~10만개 | 보통 10만+ |
| **계산 자원** | CPU 충분 | **GPU 강력 권장** |
| **학습 시간** | 분 | 시간~일 |
| **결과 해석** | 어느 정도 | **거의 불가능** |
| **잘하는 분야** | 표 데이터 | **이미지/음성/텍스트** |
| **표 데이터** | 보통 더 잘함 | 보통 비슷하거나 약간 못함 |
| **이미지/음성** | 약함 | **압도적으로 잘함** |

> **회사에서 모델 선택 가이드:**
> - 표 형태 데이터 → 일단 XGBoost. 안 되면 DL.
> - 이미지/음성/텍스트 → 처음부터 DL (또는 사전학습된 모델).

## 신경망의 종류 한눈에

이 자료에서는 가장 기본인 **MLP (Multi-Layer Perceptron)** 만 다룹니다. 하지만 신경망에는 다양한 구조가 있어요.

| 약자 | 풀이름 | 특징 / 용도 |
|------|------|------|
| **MLP** | Multi-Layer Perceptron | 일반 표 데이터 (우리가 배울 거) |
| **CNN** | Convolutional Neural Network | **이미지** 처리의 표준 |
| **RNN** | Recurrent Neural Network | 순차 데이터 (구식) |
| **LSTM** | Long Short-Term Memory | RNN의 개선 (시계열) |
| **Transformer** | (그냥 Transformer) | **현대 AI의 거의 전부.** ChatGPT, BERT, ViT 등 |
| **GAN** | Generative Adversarial Network | 이미지 생성 |
| **Diffusion** | (그냥 Diffusion) | 이미지/영상 생성 (Stable Diffusion 등) |

이 자료는 **MLP**까지 다루고, 그 이후는 [부록의 더 나아가기](../../부록/더-나아가기.md)에서 다음 학습 자료를 안내드립니다.

---

## 정리

```
신경망 = 곱하고 더하고 + 꾸불꾸불(활성화) 의 반복

      입력 → [Layer 1] → [Layer 2] → ... → [Layer N] → 출력
              w + b         w + b              w + b
              ↓             ↓                   ↓
              ReLU         ReLU                ReLU
              
학습:
  1. forward: 입력 → 출력
  2. loss: 답이 얼마나 틀렸나
  3. backward: 가중치를 어떻게 바꾸면 loss가 줄어들지
  4. step: 가중치 업데이트
  5. 반복
```

수학을 좀 더 알고 싶으시면 책 한 권쯤 보셔야 하지만, **사용하는 데에는 위 요약만으로 충분**해요. 진짜로요.
