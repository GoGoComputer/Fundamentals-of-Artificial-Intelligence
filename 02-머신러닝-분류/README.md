## 시작하기 전에 알아두실 것

### 우리는 **scikit-learn**을 씁니다

머신러닝 라이브러리에는 여러 가지가 있는데(scikit-learn, XGBoost, LightGBM, ...), **이 장에서는 `scikit-learn`(줄여서 `sklearn`)을 쓸 거예요.** 이유는:

- 가장 표준적
- 거의 모든 머신러닝 모델을 같은 패턴으로 다룰 수 있음 (`fit`, `predict`)
- 문서가 훌륭함
- Colab에 이미 설치되어 있음

다른 라이브러리들도 결국 `sklearn`의 문법을 따라가는 경우가 많아요. 한 번 익혀두면 평생 갑니다.

### 이 장은 모두 **CPU**로 돌아갑니다

머신러닝 모델은 보통 GPU 없이도 잘 돌아가요. Colab에서도 GPU 안 켜셔도 됩니다. 4장(딥러닝)부터 GPU가 필요해요.

### 코드 패턴이 거의 똑같아요

이 장에서 배우실 모든 머신러닝 코드는 다음 5단계를 따라갑니다.

```python
# 1. 데이터 불러오기
X, y = load_data()

# 2. 데이터 나누기
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.2)

# 3. 모델 만들기
model = SomeClassifier()

# 4. 학습
model.fit(X_train, y_train)

# 5. 평가
predictions = model.predict(X_test)
print("정확도:", accuracy_score(y_test, predictions))
```

진짜로 이 5줄이에요. 이게 머신러닝의 80%입니다. **모든 모델이 다 이 패턴**이에요.
