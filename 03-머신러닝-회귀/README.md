## 2장과의 관계

이 장은 [2장 (분류)](../02-머신러닝-분류/)와 **거의 쌍둥이**입니다. 코드 패턴이 똑같아요.

```python
# 2장 (분류)
from sklearn.svm import SVC
model = SVC()
model.fit(X_train, y_train)
predictions = model.predict(X_test)

# 3장 (회귀)
from sklearn.svm import SVR    # ← C가 R로 바뀜
model = SVR()
model.fit(X_train, y_train)
predictions = model.predict(X_test)
```

가장 큰 차이는 두 가지예요.

1. **`y`가 카테고리(0,1,2,...)가 아니라 연속된 숫자(1.5, 3.7, 100.0...)**
2. **평가 지표가 정확도가 아니라 MSE, RMSE 등**

이 두 가지만 익히시면 회귀는 거의 끝입니다.

## 2장을 안 보신 분에게

이 장은 [2장 분류](../02-머신러닝-분류/)와 거의 동일한 패턴을 사용해요. 다음 내용은 2장에서 자세히 설명되어 있고 여기서는 짧게만 다룹니다.

- `train_test_split` 사용법
- 정규화 (StandardScaler)
- Pipeline
- GridSearchCV / RandomizedSearchCV

가능하시면 2장을 먼저 보시는 걸 권합니다. 시간이 없다면, 막히는 곳에서 2장으로 점프하셔서 보세요.
