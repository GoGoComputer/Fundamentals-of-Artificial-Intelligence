> 분류 모델의 출력은 "확률", 학습 데이터는 "분포". ML은 사실 통계의 응용이에요.

## 5.1 평균과 분산 — 통계의 출발점

### 평균 (Mean)

다 더해서 개수로 나눈 것. 누구나 알죠.

```
데이터: 2, 4, 6, 8, 10
평균 = (2 + 4 + 6 + 8 + 10) / 5 = 30 / 5 = 6
```

수식으로:
```
평균 = (1/n) × Σᵢ xᵢ
```

기호로 평균은 `μ` (뮤) 또는 `x̄` (엑스 바).

### 분산 (Variance)

데이터가 **얼마나 흩어져 있는지**.

```
같은 평균이라도:

데이터 A: 5, 6, 7, 6, 6      (평균 6, 흩어짐 작음)
데이터 B: 0, 12, 6, 0, 12    (평균 6, 흩어짐 큼)
```

분산을 어떻게 계산할까?

```
분산 = (1/n) × Σᵢ (xᵢ - 평균)²
                    ↑
              "평균에서 얼마나 떨어져 있나" 의 제곱
```

**왜 제곱?** 양수와 음수가 섞이면 상쇄되니까. 그리고 큰 차이에 더 큰 페널티.

기호로 분산은 `σ²` (시그마 제곱).

### 표준편차 (Standard Deviation)

분산의 제곱근.

```
σ = √분산 = √((1/n) Σ (xᵢ - μ)²)
```

분산은 단위가 (원래 단위)²라 해석이 어려워요. 표준편차는 원래 단위라 직관적이에요.

**예시:** 키 데이터 (cm)의 분산은 `cm²`. 표준편차는 `cm`. 그래서 표준편차가 더 자주 쓰여요.

### Python에서

```python
import numpy as np

data = np.array([2, 4, 6, 8, 10])

print(np.mean(data))    # 6.0
print(np.var(data))     # 8.0   분산
print(np.std(data))     # 2.83  표준편차
```

### ML에서 어디?

#### 1. StandardScaler (정규화)

[3장 정규화](../../03-머신러닝-회귀/이론/05-정규화.md)에서 봤죠.

```
정규화된 값 = (원래 값 - 평균) / 표준편차
```

이렇게 하면 평균 0, 표준편차 1인 데이터로 변환돼요.

#### 2. Batch Normalization

[4장](../../04-딥러닝-분류/이론/05a-Batch-Normalization-깊게.md). 신경망 안에서 같은 일을 함.

#### 3. 평가 지표

여러 실험 결과의 평균 ± 표준편차로 보고함.

```
정확도: 92.3% ± 1.5%
```

## 5.2 확률 — 기초

### 확률이 뭐예요?

**"어떤 일이 일어날 가능성"** 을 0~1 사이 숫자로 표현한 것.

- 0 = 절대 안 일어남
- 1 = 무조건 일어남
- 0.5 = 반반

### 표기

```
P(A)
```

`A`라는 사건이 일어날 확률. 예를 들어:

```
P(동전 앞면) = 0.5
P(주사위에서 6) = 1/6 ≈ 0.167
P(내일 비) = 0.3   (날씨예보)
```

### 확률의 합

**가능한 모든 결과의 확률의 합 = 1.**

```
주사위 던지기:
P(1) + P(2) + P(3) + P(4) + P(5) + P(6) = 1/6 × 6 = 1
```

### 조건부 확률 (Conditional Probability)

**B가 일어났다는 조건 하에 A가 일어날 확률.**

```
P(A | B)    ← "B 주어졌을 때 A의 확률"
```

예시:

```
P(비 옴) = 0.3                       (그냥 내일 비 올 확률)
P(비 옴 | 흐림) = 0.7                 (흐릴 때 비 올 확률)
                  ↑
              "흐림" 이라는 조건
```

같은 사건이지만 조건에 따라 확률이 달라요.

### 독립 (Independent)

두 사건이 서로 영향 없으면 **독립**.

```
P(A | B) = P(A)    ← B가 일어나든 말든 A의 확률 같음
```

예시: 동전을 두 번 던질 때, 첫 번째 결과가 두 번째에 영향을 안 줌. 독립.

독립이면 동시에 일어날 확률은 곱.

```
P(A AND B) = P(A) × P(B)
```

### ML에서

#### 1. 분류 모델의 출력은 확률

```python
model.predict_proba(x)
# [0.05, 0.10, 0.85, 0.00, ...]   ← 합 = 1
```

가장 큰 확률의 클래스를 답으로 선택.

#### 2. 조건부 확률로 보면

분류 모델이 하는 일: `P(클래스 | 입력)` 을 추정.

```
P(고양이 | 이 사진) = 0.92
P(강아지 | 이 사진) = 0.05
P(토끼   | 이 사진) = 0.03
```

## 5.3 정규분포 (Normal Distribution)

### 직관

가장 흔하고, 가장 중요한 분포. **종 모양 곡선**.

```
       y (확률 밀도)
       ↑
       │      ___
       │     /   \
       │    /     \
       │   /       \
       │  /         \
       │ /           \
       │/             \
   ────┼───────────────────→ x
       │      ↑
       │    평균
       │
```

### 성질

- 평균 주변에 데이터가 가장 많이 모여 있음
- 평균에서 멀어질수록 데이터가 적어짐
- 좌우 대칭

### 매개변수

정규분포는 **두 개의 숫자**로 완전히 정의돼요.

- `μ` (평균): 중심 위치
- `σ` (표준편차): 퍼진 정도

```
평균은 같고 표준편차가 다른 정규분포

       │
       │   /\         좁음 (σ 작음)
       │  /  \
       │ /    \
       │/______\
       │
       │     /\\\\
       │    /    \\\\  넓음 (σ 큼)
       │   /        \\\\
       │__/____________\\\\
```

### 표기

```
X ~ N(μ, σ²)
```

읽으면: **"X는 평균 μ, 분산 σ²인 정규분포를 따른다."**

### 표준정규분포

평균 0, 표준편차 1인 정규분포. `Z ~ N(0, 1)`.

### 정규분포의 식 (참고만)

```
p(x) = (1/√(2πσ²)) × e^(-(x-μ)²/(2σ²))
```

복잡해 보이지만 핵심은 **가운데 (x=μ) 에서 가장 큰 값, 양옆은 e^(-숫자²) 로 빠르게 작아짐.**

### 68-95-99.7 규칙

기억해두면 좋아요.

```
평균 ± 1σ 안에 약 68%의 데이터
평균 ± 2σ 안에 약 95%의 데이터
평균 ± 3σ 안에 약 99.7%의 데이터
```

이상치 탐지에 자주 써요. **평균에서 3σ 이상 떨어지면 이상치 의심.**

### ML에서

#### 1. 가중치 초기화

신경망 가중치를 처음에 작은 랜덤 값으로 초기화. 보통 정규분포에서 뽑음.

```python
torch.randn(...)    # 표준 정규분포에서 샘플링
```

#### 2. 노이즈 추가 (정규화 기법)

데이터에 가우시안 노이즈 추가. 정규분포에서 뽑은 노이즈.

#### 3. 손실 함수의 가정

회귀 문제에서 MSE 손실은 **잔차가 정규분포를 따른다**는 가정과 같아요.

#### 4. 통계적 가설검정

A/B 테스트에서 두 모델 차이가 유의한지.

## 5.4 다른 분포들 (간단히)

ML에서 가끔 만나는 분포들.

### 균등분포 (Uniform)

모든 값이 같은 확률.

```
       p
       │
       │_______________
       │
       │
  ─────┼───────────────→ x
       a               b
```

`np.random.uniform(0, 1)` 같이 무작위 수 만들 때.

### 베르누이 (Bernoulli)

0 또는 1 둘 중 하나. 동전 던지기.

```
P(X = 1) = p
P(X = 0) = 1 - p
```

이진 분류 라벨.

### 이항 (Binomial)

베르누이를 n번 반복했을 때 1이 나온 횟수.

### 푸아송 (Poisson)

단위 시간 내 일어난 사건 수. 콜센터 분당 전화 수 같은.

이 정도만 알아두시면 충분해요.

## 5.5 베이즈 정리 (Bayes' Theorem)

확률에서 가장 중요한 정리. 그리고 ML의 한 갈래(베이지안 ML)의 출발점.

### 식

```
P(A | B) = P(B | A) × P(A) / P(B)
```

말로: **"B 주어졌을 때 A의 확률 = (A 주어졌을 때 B의 확률 × A의 확률) / B의 확률."**

### 직관: 검사 문제

병에 걸렸는지 검사한다고 해 봐요.

```
병에 진짜 걸린 비율: 1% → P(병) = 0.01

검사의 정확도:
- 병에 걸린 사람이 양성 나올 확률: 99% → P(양성|병) = 0.99
- 안 걸린 사람이 양성 나올 확률 (오진): 5% → P(양성|건강) = 0.05
```

질문: **양성이 나온 사람이 진짜 병에 걸렸을 확률은?**

직관적으로는 99%? 베이즈 정리로 풀어 봐요.

```
P(병|양성) = P(양성|병) × P(병) / P(양성)

P(양성) = P(양성|병)×P(병) + P(양성|건강)×P(건강)
       = 0.99 × 0.01 + 0.05 × 0.99
       = 0.0099 + 0.0495
       = 0.0594

P(병|양성) = 0.99 × 0.01 / 0.0594
         = 0.0099 / 0.0594
         ≈ 0.167
```

**16.7%!**

검사 정확도가 99%여도, 진짜 병에 걸린 사람은 양성 환자의 17%밖에 안 돼요. 충격적이죠?

이유: 진짜 병에 걸린 사람이 너무 적기 때문에, 오진이 쌓여서 양성 환자의 대부분이 건강한 사람이에요.

이걸 **사전 확률(prior)** 의 중요성이라고 해요. 의료/법률에서 매우 중요한 개념.

### ML에서

#### 1. 나이브 베이즈 분류기

스팸 필터의 고전. 문서가 스팸일 확률을 베이즈로 계산.

```
P(스팸 | 단어들) = P(단어들 | 스팸) × P(스팸) / P(단어들)
```

#### 2. 베이지안 신경망

가중치를 단일 값이 아니라 **분포**로 봐요. 불확실성을 표현 가능.

#### 3. A/B 테스트

베이지안 A/B 테스트가 점점 표준이 되어 가요.

## 5.6 가능도 (Likelihood) — 한 단어로

> "이 데이터가 나올 확률이, 이런 모델 (또는 파라미터) 일 때 얼마인가?"

```
L(θ | 데이터) = P(데이터 | θ)
                ↑
           파라미터 θ 가정 하에
```

`L`이 클수록 → 그 파라미터가 데이터를 잘 설명.

### 최대 가능도 추정 (Maximum Likelihood Estimation, MLE)

가능도를 가장 크게 만드는 파라미터를 찾자!

수많은 ML 모델이 사실은 MLE에서 도출돼요.

- **선형 회귀의 MSE 손실** = 잔차가 정규분포라고 가정한 MLE
- **로지스틱 회귀의 cross-entropy** = 베르누이라고 가정한 MLE

깊이 안 들어가도, **"손실 함수에는 통계적 의미가 있다"** 는 것만 기억하세요.

## 5.7 정리

```python
import numpy as np

# 평균, 분산, 표준편차
np.mean(data)
np.var(data)
np.std(data)

# 정규분포에서 샘플링
np.random.normal(loc=0, scale=1, size=100)
torch.randn(100)    # PyTorch

# 균등분포
np.random.uniform(0, 1, size=100)

# 정규화 (StandardScaler 가 하는 일)
data_normalized = (data - np.mean(data)) / np.std(data)
```

### ML 식 다시 읽기

```
StandardScaler: x_new = (x - μ) / σ
```
"새 값 = (원래 값 - 평균) / 표준편차."

```
Gaussian noise: ε ~ N(0, σ²)
```
"잡음 ε은 평균 0, 분산 σ²인 정규분포를 따른다."

```
P(y | x) — 분류 모델의 출력
```
"입력 x 주어졌을 때 라벨 y의 확률."

```
P(y|x) = P(x|y) × P(y) / P(x)   (베이즈)
```
"x 주어졌을 때 y의 확률 = (y 주어졌을 때 x의 확률 × y의 확률) / x의 확률."

다 읽혀요. 진짜로요.
