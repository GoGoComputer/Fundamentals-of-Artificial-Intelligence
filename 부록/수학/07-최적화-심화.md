# 7. 최적화 심화 — 라그랑주, 볼록 최적화, Newton's Method

> [6장](06-ML에-적용하기.md)에서 경사하강법을 봤죠. 이 장은 그것을 넘어서는 최적화 기법들.
> SVM의 수학적 기초(라그랑주), 더 빠른 수렴 방법(Newton), 이론적 보장(볼록).

---

## 7.1 최적화 문제의 큰 그림

### 일반 형태

```
minimize    f(x)        ← 최소화하고 싶은 것 (목적 함수, objective)

subject to  gᵢ(x) ≤ 0   ← 부등식 제약
            hⱼ(x) = 0   ← 등식 제약
```

### ML에서

- `f(x)` = 손실 함수
- `x` = 가중치 (파라미터)
- 제약 = 보통 없음 (있어도 정규화로 흡수)

---

## 7.2 볼록 최적화 (Convex Optimization)

### 볼록 함수 (Convex Function)

```
함수 f가 볼록 ⇔ 함수 위 어떤 두 점을 잇는 선분이 항상 함수 위에 있음
```

```
볼록 함수:                    비볼록 함수:
y                             y
│   ╱                         │  ╱╲    ╱
│  ╱                          │ ╱  ╲  ╱
│ ╱                           │╱    ╲╱
│╱   _____   ← 두 점 잇는 선분  │
└──────────→ x                 └──────────→ x
   함수 위에 있음               선분이 함수 아래로 내려감
```

### 왜 중요?

**볼록 함수는 local minimum = global minimum.** 즉, 한 번 골짜기 찾으면 거기가 진짜 최저점.

```
볼록 함수의 학습:
     /
    /
   /
  ╲
   ╲___╳___╱   ← 어디서 시작해도 같은 점에 도달
       
비볼록 함수의 학습 (신경망):
     ╱╲    ╱╲
    ╱  ╲  ╱  ╲
   ╲    ╳╲___╲╱      ← 시작 위치에 따라 다른 점
   여기서 멈출 수도, 더 좋은 데가 있을 수도
```

### ML에서?

| 모델 | 손실 함수 | 볼록? |
|------|---------|------|
| 선형 회귀 (MSE) | 볼록 | ✅ Global minimum 보장 |
| 로지스틱 회귀 (CE) | 볼록 | ✅ |
| SVM | 볼록 | ✅ |
| **신경망** | **비볼록** | ❌ Local minimum 가능 |

신경망은 비볼록이지만, 실용적으로는 좋은 local minimum을 자주 찾을 수 있어요. 그게 deep learning이 동작하는 이유 중 하나.

### 볼록 함수의 미분 조건

```
2차 미분 ≥ 0 (모든 곳에서)  →  볼록
```

`f(x) = x²` 의 2차 미분 = 2 > 0 → 볼록 ✓

### 다변수 — Hessian이 양정치

`H` 가 양정치(positive definite)면 함수가 볼록.

→ 볼록 최적화 이론은 **Hessian의 성질**에 매우 의존.

---

## 7.3 Newton's Method — 2차 최적화

경사하강법은 1차 미분만 씀. **Newton's Method는 2차 미분(Hessian)도 씀** → 이론적으로 더 빠름.

### 1차원 (단변수)

```
x_new = x_old - f'(x) / f''(x)
              ↑
         학습률 = f''(x)의 역수
```

해석: 곡률(`f''`)이 크면 작은 걸음, 작으면 큰 걸음.

### 다차원

```
x_new = x_old - H⁻¹ × ∇f
              ↑
        Hessian의 역행렬
```

### 직관

```
경사하강법: 기울기만 보고 한 발자국
Newton:    기울기 + 곡률 보고 한 번에 골짜기로 점프
```

### 비유

여러분이 안개 낀 산에서 내려간다고 해 봐요.
- **경사하강법**: 발 디딘 곳의 기울기만 보고 한 발자국씩
- **Newton**: 주변의 기울기 변화도 보고, "골짜기까지 정확히 X 발자국 가야겠네" 계산하고 한 번에

### 단점

- **Hessian이 너무 큼**. 신경망은 가중치가 백만 개 → Hessian은 백만×백만 행렬. 메모리/계산 폭발.
- **역행렬 계산이 비쌈** O(n³)
- **비볼록 함수에서는 잘못 이끌 수 있음** (안장점에 갇힘)

### 그래서 ML에서는?

큰 신경망에는 Newton 안 씀. 대신 **근사 방법**.

#### Quasi-Newton: BFGS / L-BFGS

Hessian을 직접 안 구하고 근사. 작은 모델에서 sklearn이 가끔 사용.

#### Adam — 일종의 적응적 1차 방법

각 가중치마다 학습률을 조정. Newton의 효과를 일부 흉내.

## 7.4 라그랑주 승수법 (Lagrange Multipliers)

**제약 있는 최적화** 푸는 방법. **SVM의 수학적 기초.**

### 등식 제약 문제

```
minimize    f(x)
subject to  g(x) = 0
```

### 라그랑주안 (Lagrangian)

```
L(x, λ) = f(x) - λ × g(x)
              ↑
         라그랑주 승수
```

### 풀이 조건

`L`을 `x`와 `λ`에 대해 모두 편미분 = 0.

```
∂L/∂x = ∇f - λ∇g = 0   →  ∇f = λ∇g
∂L/∂λ = -g(x) = 0      →  g(x) = 0  (제약 만족)
```

### 직관

> 제약에 닿은 상태에서, 목적 함수의 **그래디언트가 제약의 그래디언트와 평행**해야 극값.

### 예시

`x² + y²` 를 최소화, 단 `x + y = 2`.

```
L = x² + y² - λ(x + y - 2)

∂L/∂x = 2x - λ = 0  →  x = λ/2
∂L/∂y = 2y - λ = 0  →  y = λ/2
∂L/∂λ = -(x + y - 2) = 0  →  x + y = 2

→ λ/2 + λ/2 = 2  →  λ = 2  →  x = y = 1
```

답: `(1, 1)`. 거기서 `x²+y² = 2`.

### KKT 조건 (Karush-Kuhn-Tucker)

부등식 제약까지 포함. 라그랑주의 일반화.

```
minimize    f(x)
subject to  gᵢ(x) ≤ 0
            hⱼ(x) = 0
```

KKT 조건:
1. **Stationarity**: `∇f + Σ μᵢ∇gᵢ + Σ λⱼ∇hⱼ = 0`
2. **Primal feasibility**: `gᵢ ≤ 0, hⱼ = 0`
3. **Dual feasibility**: `μᵢ ≥ 0`
4. **Complementary slackness**: `μᵢ × gᵢ = 0`

복잡해 보이지만, **볼록 문제에서는 이 조건이 최적의 필요충분조건**.

### ML에서? — SVM의 진짜 모습

SVM은 사실 라그랑주로 푸는 제약 최적화 문제예요.

```
minimize  (1/2)||w||²
subject to  yᵢ(w·xᵢ + b) ≥ 1  (모든 i)
```

라그랑주안:
```
L = (1/2)||w||² - Σᵢ αᵢ × [yᵢ(w·xᵢ + b) - 1]
                    ↑
              SVM의 라그랑주 승수
```

### Dual Problem (쌍대 문제)

라그랑주안을 `α` 에 대해 변형하면 SVM의 **쌍대 문제**가 나오는데:

```
maximize  Σᵢ αᵢ - (1/2) Σᵢⱼ αᵢαⱼ yᵢyⱼ (xᵢ·xⱼ)
                                      ↑
                                   내적!
```

여기서 `(xᵢ·xⱼ)` 가 핵심. **이걸 커널 함수로 바꾸면 비선형 SVM**이 돼요.

```
xᵢ·xⱼ → K(xᵢ, xⱼ)   (커널 트릭)
```

이게 [2.4a SVM 깊게](../../02-머신러닝-분류/이론/04a-SVM-깊게.md) 의 진짜 수학적 기초.

### Support Vector의 정체

KKT의 complementary slackness:
```
αᵢ × [yᵢ(w·xᵢ + b) - 1] = 0
```

즉:
- `αᵢ > 0` 인 점들: **마진에 정확히 닿은 점들** = Support Vectors
- `αᵢ = 0` 인 점들: 마진에서 떨어진 점들 (영향 X)

그래서 **SVM은 일부 점들(서포트 벡터)만으로 결정 경계를 정의**해요. 효율적.

## 7.5 경사하강법의 변형들 — 최적화 관점

[4.4a](../../04-딥러닝-분류/이론/04a-경사하강법-깊게.md)에서 본 옵티마이저들을 최적화 관점에서.

### Momentum

이전 그래디언트를 누적. **언덕에서 굴러가는 공.**

```
v ← βv + ∇L
w ← w - η × v
```

수학적으로 **Heavy Ball Method** 의 일종.

### Adam — Adaptive Moment Estimation

```
m ← β₁ m + (1-β₁) ∇L          (1차 모멘트, 평균)
v ← β₂ v + (1-β₂) ∇L²         (2차 모멘트, 분산)
ŵ ← m / √(v + ε)               (정규화)
w ← w - η × ŵ
```

각 가중치마다 다르게 학습률 조정. **Newton의 일종의 1차 근사.**

### Trust Region Methods

"이 영역 안에서만 step을 이끌어라" — 안전하게.
대규모 신경망엔 안 쓰지만 알아두면 좋음.

## 7.6 일반화 — 볼록 vs 비볼록

### 볼록 최적화 (Convex Optimization)

```
✅ Global minimum 보장
✅ 알고리즘이 정확하면 정답 찾음
✅ 이론적으로 깨끗함
✅ 풀 수 있는 도구 많음 (cvxpy 등)

해당 모델: 선형 회귀, 로지스틱 회귀, SVM
```

### 비볼록 최적화 (Non-convex)

```
❌ Local minimum 가능
❌ Saddle point 가능
❌ 시작점에 의존
❌ 이론적 보장 약함

✅ 표현력 매우 강함
✅ 신경망의 비밀

해당 모델: 신경망 (거의 모든 DL 모델)
```

신경망이 비볼록인데도 잘 동작하는 이유는 깊은 미스터리 (활발히 연구 중). 경험적 규칙들 (good initialization, batch norm, etc.) 이 도움.

## 7.7 정리

```python
# 일반 최적화 패턴
import scipy.optimize as opt

# 1차 방법 (경사하강 같은)
result = opt.minimize(f, x0, jac=df, method='BFGS')   # Quasi-Newton

# 라그랑주 / KKT 풀이
result = opt.minimize(f, x0, constraints={'type':'eq', 'fun': g})

# 볼록 최적화 전용 (cvxpy)
import cvxpy as cp
x = cp.Variable(n)
problem = cp.Problem(cp.Minimize(f(x)), [g(x) <= 0])
problem.solve()
```

### ML 식 다시 읽기

```
Newton: x ← x - H⁻¹ ∇f
```
"x 새 값 = x 기존 값 - 헤시안 역행렬 × 그래디언트."

```
Lagrangian: L = f(x) - λ × g(x)
```
"라그랑주안 = 목적 함수 - 라그랑주 승수 × 제약 함수."

```
SVM dual: max Σαᵢ - (1/2) Σ αᵢαⱼ yᵢyⱼ K(xᵢ,xⱼ)
```
"SVM의 쌍대 문제: (모든 α의 합) - (1/2) × (모든 i,j의 αᵢ αⱼ yᵢ yⱼ K(xᵢ,xⱼ)의 합)을 최대화."

3. SVM에서 support vector는 KKT 조건의 어떤 조건과 관련?
4. 라그랑주 승수법은 어떤 종류의 문제에 쓰이나요?
5. 신경망의 손실 함수는 볼록인가요?

<details>
<summary>정답</summary>

1. Local minimum = Global minimum. 한 번 골짜기 찾으면 진짜 최저점.
2. Hessian이 너무 커서 (n×n 행렬) 메모리/계산 비용 폭발.
3. **Complementary slackness**: `αᵢ × [yᵢ(w·xᵢ+b)-1] = 0`. αᵢ > 0 인 점들이 support vector.
4. **제약 있는 최적화**. 등식 제약 (라그랑주) 또는 부등식 제약 (KKT 일반화).
5. **비볼록**. 그래서 local minimum, saddle point 문제가 있고, 학습이 어려움.
