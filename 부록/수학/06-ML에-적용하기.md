# 6. ML에 적용하기 — 모든 것을 합치기

> 1~5장에서 배운 수학을 ML의 핵심 부분에 적용해 봅시다.
> 본문(0~6장)에서 만났던 식들이 이제 다 읽혀야 해요.

---

## 6.1 경사하강법 — 다시 한 번 깊게

### 식

```
w_new = w_old - η × ∇L
        ↑              ↑
     기존 가중치    그래디언트
```

각 기호:
- `η` (에타): 학습률 (보통 0.001 ~ 0.1)
- `∇L`: 손실 L의 그래디언트 = 모든 편미분의 모음
- `-` 부호: 기울기 반대 방향으로 가야 손실이 감소

### 한 가중치 단위로 풀어 보면

```
w₁ ← w₁ - η × ∂L/∂w₁
w₂ ← w₂ - η × ∂L/∂w₂
...
wₙ ← wₙ - η × ∂L/∂wₙ
```

신경망에 가중치 백만 개면 백만 개를 동시에 이렇게 업데이트.

### PyTorch에서

```python
optimizer = torch.optim.SGD(model.parameters(), lr=0.01)

# 학습 루프
loss.backward()      # ∇L 자동 계산
optimizer.step()      # w ← w - lr × grad 자동 실행
```

`SGD` 한 줄이 위 식 전체를 의미해요.

### Adam은 뭐가 다른가?

기본 SGD는 그냥 그래디언트 따라가요. **Adam은 과거 그래디언트들의 평균과 분산을 활용해요.**

```
m ← β₁ × m + (1-β₁) × ∇L     (1차 모멘트, 평균 같은 것)
v ← β₂ × v + (1-β₂) × ∇L²    (2차 모멘트, 분산 같은 것)
w ← w - η × m / (√v + ε)
```

복잡해 보이지만, 핵심은 **"각 가중치마다 다르게 학습률 조정"** 이에요. ([4장 보강](../../04-딥러닝-분류/이론/04a-경사하강법-깊게.md))

## 6.2 손실 함수 — 수학으로 풀어보기

### MSE (Mean Squared Error)

```
MSE = (1/n) × Σᵢ (yᵢ - ŷᵢ)²
```

읽기:
- `(1/n)`: 샘플 수로 나누기 (평균)
- `Σᵢ`: 모든 샘플에 대해 더하기
- `(yᵢ - ŷᵢ)²`: 실제값과 예측값의 차이의 제곱

`y`는 실제, `ŷ`(와이햇) 은 예측. 이 표기법은 ML 표준이에요.

### MSE 의 미분

`∂L/∂ŷᵢ = ?`

`L = (1/n) Σ (yᵢ - ŷᵢ)²` 를 `ŷᵢ`로 편미분.

```
∂L/∂ŷᵢ = (1/n) × 2 × (yᵢ - ŷᵢ) × (-1)
        = -2(yᵢ - ŷᵢ) / n
        = 2(ŷᵢ - yᵢ) / n
```

복잡해 보이지만, 핵심은 **"예측이 실제보다 크면 양수 (즉, 줄여야 함), 작으면 음수 (즉, 키워야 함)."** 이게 학습 방향을 알려줘요.

### Cross-Entropy

```
CE = -Σᵢ yᵢ × log(pᵢ)
```

여기서:
- `yᵢ`: 정답 라벨 (one-hot 또는 정수)
- `pᵢ`: 모델의 예측 확률 (softmax 통과 후)

### One-hot 인코딩이 뭐예요?

3개 클래스 (강아지/고양이/토끼) 분류.

```
정답이 "고양이" 라면:
y = [0, 1, 0]    ← 정답 위치만 1, 나머지 0
```

이게 **one-hot 인코딩**이에요. 정답 클래스만 1.

### CE 식을 풀어보면

```
y = [0, 1, 0]                  ← 정답 (고양이)
p = [0.1, 0.7, 0.2]            ← 모델 예측

CE = -(0 × log 0.1 + 1 × log 0.7 + 0 × log 0.2)
   = -(0 + log 0.7 + 0)
   = -log 0.7
   ≈ 0.357
```

핵심: **정답 위치의 예측 확률만 보면 되고, 나머지는 0으로 곱해져 사라져요.**

```
CE = -log(p_정답)
```

이렇게 단순해져요.

### 직관

**`-log(p)` 의 모양:**

```
       y = -log(p)
       │
       │ ▏
       │ ▏
       │  ▏
       │   \
       │    \
       │     \____
       │          ─____
   ────┼─────────────────────→ p
       0   0.5     1
       
       p가 1에 가까울수록 0에 가까움 (정답 잘 맞춤)
       p가 0에 가까울수록 무한대 (정답 완전 틀림)
```

정답 확률이 높으면 손실이 작고, 낮으면 손실이 무한대로 커져요. **틀린 것에 매우 큰 페널티**를 주는 손실이에요.

### Cross-Entropy의 미분

생각보다 단순해요.

```
∂CE/∂(W의 마지막 층) = (p - y)   ← softmax 나온 확률에서 정답 빼기
```

예측이 정답에 가까우면 그래디언트 작고, 멀면 큼. **자연스러운 학습 신호.** Softmax + CE 조합이 좋은 이유 중 하나예요.

## 6.3 Sigmoid와 Softmax — 활성화 함수의 수학

### Sigmoid

```
σ(x) = 1 / (1 + e⁻ˣ)
```

읽기:
- `e⁻ˣ`: 자연상수의 -x 제곱 (음수일수록 큰 값)
- 분모가 `1 + e⁻ˣ` 라서, 결과는 항상 0과 1 사이

### 값 확인

```
σ(0) = 1 / (1 + 1) = 0.5      (정확히 가운데)
σ(1) = 1 / (1 + 0.368) ≈ 0.73
σ(-1) = 1 / (1 + 2.718) ≈ 0.27
σ(10) ≈ 0.99995              (큰 양수 → 1에 가까움)
σ(-10) ≈ 0.00005             (큰 음수 → 0에 가까움)
```

S자 모양 곡선이라 sigmoid라는 이름.

### Sigmoid의 미분

```
σ'(x) = σ(x) × (1 - σ(x))
```

신기하게 자기 자신으로 표현돼요. 그래서 역전파에서 매우 효율적이에요.

### Softmax

여러 클래스의 점수를 **합이 1인 확률들**로 변환.

```
softmax(zᵢ) = e^zᵢ / Σⱼ e^zⱼ
```

### 예시

3개 클래스의 점수 (logit) `z = [2.0, 1.0, 0.1]`.

```
e^2.0 = 7.389
e^1.0 = 2.718
e^0.1 = 1.105

합 = 7.389 + 2.718 + 1.105 = 11.212

softmax = [7.389/11.212, 2.718/11.212, 1.105/11.212]
        = [0.659, 0.242, 0.099]

확인: 0.659 + 0.242 + 0.099 = 1.000 ✅
```

큰 점수가 큰 확률, 작은 점수가 작은 확률. 합은 1.

### 왜 e (자연상수)?

지수 함수가 항상 양수이고, 미분이 깔끔해서 (자기 자신).
또한 `e^x`는 큰 차이를 더 크게 만들어요. 점수 차이를 강조해서 "확신 있는 답" 을 주는 효과.

## 6.4 정규화 (Regularization) 의 수학

### L2 Regularization (Ridge)

```
손실 = MSE + λ × Σᵢ wᵢ²
                  ↑
              가중치 제곱의 합
```

`λ` (람다) 는 정규화 강도. 보통 0.001 ~ 0.1.

### 왜 효과가 있어요?

`Σwᵢ²` 가 커지면 손실이 커져요. 그래서 학습은 **`Σwᵢ²` 를 작게 유지하면서 MSE도 작게**.

→ 가중치들이 너무 커지지 못함 → 모델이 부드러워짐 → 과적합 감소.

### 미분으로 보면

```
∂L/∂w = ∂MSE/∂w + 2λw
```

`+2λw` 가 추가됐어요. 이건 **"가중치를 0 쪽으로 살짝 끌어당기는 힘"** 이에요.

PyTorch에서 `weight_decay=0.01` 이 정확히 이 `2λ` 역할.

### L1 Regularization (Lasso)

```
손실 = MSE + λ × Σᵢ |wᵢ|
                  ↑
              가중치 절댓값의 합
```

차이: `wᵢ²` 대신 `|wᵢ|`. 

L1의 특징: **일부 가중치를 정확히 0으로** 만듦. 자동 특성 선택. ([3.5](../../03-머신러닝-회귀/이론/05-정규화.md))

## 6.5 역전파 (Backpropagation) — 체인룰의 응용

### 신경망 한 층 다시

```
h = σ(Wx + b)
```

### 손실까지 가는 흐름

```
입력 x → Wx + b → σ(.) → 다음 층 → ... → 손실 L
```

### 역전파 — 손실에서 거꾸로

손실 `L`을 첫 층의 가중치 `W₁`에 대해 편미분하려면? **체인룰**.

```
∂L/∂W₁ = ∂L/∂h_N × ∂h_N/∂h_{N-1} × ... × ∂h₂/∂h₁ × ∂h₁/∂W₁
         └── 출력층 ──┘  └── 중간층들 ──┘     └ 첫 층 ┘
```

각 층을 거꾸로 거슬러 올라가며 미분을 곱함.

### 한 층의 역전파 (간단 버전)

`h = σ(Wx + b)` 의 역전파:

```
∂L/∂W = ∂L/∂h × σ'(Wx+b) × x
                ↑
           출력 층에서 온 그래디언트

∂L/∂b = ∂L/∂h × σ'(Wx+b)

∂L/∂x = ∂L/∂h × σ'(Wx+b) × W   ← 이전 층으로 전달!
```

마지막 줄이 핵심. **그래디언트가 한 층씩 거슬러 흘러요.** 그래서 **"역전파"** 라고 하는 거예요.

### PyTorch 마법

```python
loss.backward()
```

이 한 줄이 위의 어마어마한 체인룰 계산을 자동으로. **autograd**가 모든 텐서 연산을 추적해서, 미분 그래프를 만들고, 거꾸로 흘려요.

직접 손으로 구현하려면 며칠 걸리는 일을 한 줄로. 위대해요.

## 6.6 Softmax + Cross-Entropy — 가장 자주 만나는 조합

분류 신경망의 마지막 두 단계.

```
점수(logit) → softmax → 확률 → cross-entropy → 손실
```

### 식 풀어 쓰기

```
점수: z = (z₁, z₂, ..., z_K)         ← 마지막 층 출력 (활성화 X)

확률: pᵢ = e^zᵢ / Σⱼ e^zⱼ            ← softmax

손실: L = -Σᵢ yᵢ × log(pᵢ)            ← cross-entropy
        = -log(p_정답)                ← one-hot이라 단순화
```

### 미분 (마법 같은 결과)

이 둘을 같이 미분하면:

```
∂L/∂zᵢ = pᵢ - yᵢ
```

**이렇게 깔끔해요.** "예측 확률 - 정답"

이게 너무 깔끔해서 PyTorch의 `nn.CrossEntropyLoss`가 **softmax를 안에 자동으로 포함**시켰어요. 우리가 마지막 층에 활성화를 안 주는 이유.

```python
# 모델 — 마지막 층은 logit 그대로
class Model(nn.Module):
    def forward(self, x):
        ...
        return self.last_fc(x)    # 활성화 없음!

loss_fn = nn.CrossEntropyLoss()    # 안에서 softmax 적용 + CE 계산
```

## 6.7 손실의 모양 — 직관

### 파라미터 두 개의 손실

가중치가 `w₁`, `w₂` 두 개일 때, 손실 `L(w₁, w₂)` 의 모양은 3차원.

```
   L (높이)
       ↑
       │      ___
       │    /     \         손실의 "지형"
       │   /  골짜기 \
       │  /          \
       │ /            \
       └────────────────→ w₂
       w₁
```

학습 = **이 지형을 굴러 내려가며 가장 낮은 곳 (골짜기)을 찾는 일.**

### 진짜 신경망은?

가중치가 백만 개면 백만 차원 지형이에요. 상상도 못 할 모양이지만, 수학적으로 같은 일을 해요.

### 등고선 (contour) 으로 보기

3D를 못 그리니, 같은 손실을 가진 점들을 등고선으로 표시.

```
       w₂
       ↑
       │      ╱⌒⌒⌒╲
       │    ╱ ╱⌒╲╲ ╲
       │   │ │ ●─이 안이 골짜기
       │    ╲ ╲___╱╱
       │      ╲___╱
       └─────────────→ w₁
```

경사하강법은 매 step마다 **등고선의 수직 방향**으로 한 발자국. (그래디언트는 등고선에 수직이에요.)

## 6.8 정리 — 본문 식들 다시 읽기

이제 본문 어디든 펼치셔서 모든 식을 읽어 보세요.

### 4장 학습 식

```
w_new = w_old - lr × ∂L/∂w
```
"새 가중치 = 기존 가중치 - 학습률 × 손실의 가중치에 대한 편미분."

### 5장 정규화 식

```
손실 = MSE + λ × Σwᵢ²
```
"손실 = 평균제곱오차 + 람다 × 모든 가중치의 제곱의 합."

### 신경망 한 층

```
y = σ(Wx + b)
```
"출력 = 활성화함수( 가중치 행렬 × 입력 벡터 + 편향 벡터 )."

### Softmax + CE

```
p = softmax(z)
L = -Σᵢ yᵢ × log(pᵢ)
```
"확률 = softmax(점수). 손실 = -(정답 × 예측확률의 로그)의 합."

다 읽혀요. 정말 다 읽혀요.

## 자가 진단 — 종합

다음 식들을 한국어로 풀어 읽어 보세요.

1. `w ← w - η × ∂L/∂w`
2. `L = -(1/n) Σᵢ [yᵢ log(p̂ᵢ) + (1-yᵢ) log(1-p̂ᵢ)]`  (이진 분류)
3. `p̂ᵢ = e^zᵢ / Σⱼ e^zⱼ`
4. `Cov(X, Y) = E[(X - μ_X)(Y - μ_Y)]`
5. `y = ReLU(W₂ × ReLU(W₁ × x + b₁) + b₂)`

<details>
<summary>풀이 (참고)</summary>

1. "새 가중치 = 기존 가중치 - 학습률 × 손실의 가중치 편미분."
2. "손실 = 마이너스 (1/n) × 모든 i에 대한 [실제 라벨 × log 예측확률 + (1-실제) × log(1-예측)]." → 이진 cross-entropy
3. "i번째 클래스의 예측확률 = e의 zᵢ 제곱 / 모든 j의 e의 zⱼ 제곱의 합." → softmax
4. "X와 Y의 공분산 = E[(X - X의 평균)(Y - Y의 평균)]." → 공분산 정의
5. "출력 = ReLU( W₂ × ReLU( W₁ × 입력 + 편향₁ ) + 편향₂ )." → 2층 신경망

다 읽히셨다면, 본문 0~6장의 모든 식이 이제 읽힐 거예요.

</details>
