## 5a.1 공분산과 상관계수

두 변수의 **관계**를 측정.

### 공분산 (Covariance)

```
Cov(X, Y) = E[(X - μ_X)(Y - μ_Y)]
```

직관:
- 양수: X 크면 Y도 큼 (양의 상관)
- 음수: X 크면 Y는 작음 (음의 상관)
- 0: 관련 없음 (독립이면 0, 역은 아님)

### 상관계수 (Correlation Coefficient)

공분산을 표준편차로 나눠서 -1~1 사이 정규화.

```
ρ(X, Y) = Cov(X, Y) / (σ_X × σ_Y)
```

```
ρ = 1   →  완벽한 양의 선형 관계
ρ = -1  →  완벽한 음의 선형 관계
ρ = 0   →  선형 관계 없음
```

### Python

```python
import numpy as np
import pandas as pd

# pandas로 한 번에
df = pd.DataFrame({'X': X_data, 'Y': Y_data})
print(df.corr())    # 상관계수 행렬

# 또는 numpy
np.cov(X, Y)       # 공분산
np.corrcoef(X, Y)  # 상관계수
```

### ML에서?

- **특성 선택**: 타겟과 강한 상관 있는 특성 골라내기
- **다중공선성 진단**: 특성들끼리 상관 확인 ([3.7 회귀 체크리스트](../../03-머신러닝-회귀/이론/07-현업-체크리스트.md))
- **PCA의 출발점**: 공분산 행렬

### ⚠️ 주의: 상관 ≠ 인과

`ρ = 0.9` 라도 X가 Y의 **원인**이라고 단정할 수 없어요. ML 분석에서 자주 헷갈리는 함정.

## 5a.2 중심극한정리 (Central Limit Theorem, CLT)

통계학의 가장 중요한 정리. **모든 통계적 추론의 기둥.**

### 정리

> **어떤 분포이든**, 그 분포에서 뽑은 **표본의 평균**은,
> 표본 크기가 크면 **정규분포**에 가까워진다.

### 직관

```
원본 데이터 분포 (예: 매우 비대칭):
       █
       █
       █▄
       ███▄▄▄
       █████████___

→ 표본 100개씩 뽑아서 평균 → 평균들의 분포

   ___▄███▄___
  ↑           ↑
  정규분포!
```

원본이 어떻게 생겼든, 표본 평균의 분포는 정규분포!

### 정확히

`X₁, X₂, ..., Xₙ` 이 평균 `μ`, 분산 `σ²` 인 분포에서 독립적으로 뽑혔다면:

```
X̄ = (X₁ + X₂ + ... + Xₙ) / n   ← 표본 평균

n이 크면:
X̄ ~ N(μ, σ²/n)
```

표본 평균의 분산은 `σ²/n`. **n 커지면 분산 줄어듦** = 추정이 더 정확해짐.

### 왜 중요?

1. **표본으로 모집단 추정 가능** (모집단 분포 몰라도)
2. **신뢰 구간 계산** 가능
3. **가설 검정**의 이론적 기초

### ML에서?

- **모델 평가의 신뢰성**: 5-fold CV 결과의 평균이 진짜 성능에 가까움
- **A/B 테스트**: 사용자 행동 평균이 정규분포에 가까움
- **앙상블 효과**: 여러 모델 예측의 평균이 더 안정 (CLT의 일종)

## 5a.3 추정 (Estimation)

### 모집단 vs 표본

```
모집단 (Population): 우주 전체
   ↓ (일부만)
표본 (Sample): 우리가 가진 데이터
```

진짜 알고 싶은 건 모집단 (예: "전 세계 사용자의 평균 클릭률"). 하지만 다 못 보니 표본으로 추정.

### 점추정 (Point Estimation)

하나의 값으로 추정.

```
표본 평균 X̄ → 모집단 평균 μ 의 추정값
표본 분산 s² → 모집단 분산 σ² 의 추정값
```

### 표본 분산의 함정 — n-1 로 나누기

```
표본 분산: s² = (1/(n-1)) × Σ (xᵢ - X̄)²
                    ↑
                  n 아니라 n-1 !
```

**왜 n-1?** 표본 평균을 사용했기 때문에 자유도(degrees of freedom)가 1 줄어요. 자세한 증명은 깊지만, 결과만 외우세요. **표본의 분산은 n-1로.**

```python
np.var(data, ddof=0)    # n으로 나눔 (모집단 분산 가정)
np.var(data, ddof=1)    # n-1로 나눔 (표본 분산, 권장)
```

### 구간 추정 (Interval Estimation)

"이 범위 안에 진짜 값이 95% 확률로 있어요" 라는 식.

### 95% 신뢰 구간

```
X̄ ± 1.96 × (σ/√n)
   ↑       ↑
   표본평균  표준오차
```

`1.96` 은 표준정규분포에서 **양 끝 2.5% 자르는 z 값.**

```python
from scipy.stats import norm

mean = np.mean(data)
se = np.std(data, ddof=1) / np.sqrt(len(data))    # 표준오차

ci_95 = (mean - 1.96 * se, mean + 1.96 * se)
print(f"95% CI: {ci_95}")
```

### ML에서?

- **모델 정확도 보고**: "정확도 92.3% ± 1.5%" 의 ±는 신뢰구간
- **A/B 테스트의 결과 해석**
- **Cross-Validation 결과의 평균과 분산**

## 5a.4 가설 검정 (Hypothesis Testing)

"두 모델의 차이가 우연인가, 진짜인가?" 같은 질문에 답하는 도구.

### 절차

1. **귀무가설 (H₀)**: "차이 없음" 같은 기본 가정
2. **대립가설 (H₁)**: "차이 있음"
3. **검정 통계량 계산**
4. **p-값 (p-value) 계산**
5. **유의수준 α (보통 0.05) 와 비교**

### p-값의 의미

> "귀무가설이 사실이라고 가정할 때, **이런 데이터(또는 더 극단적인)가 나올 확률**."

- `p < 0.05` → 귀무가설 기각, "유의하다"
- `p ≥ 0.05` → 귀무가설 못 기각, "유의하지 않다"

### t-검정 (Two-sample t-test) — 두 그룹 평균 비교

```python
from scipy.stats import ttest_ind

# 두 모델의 5번 실험 결과
model_A_scores = [0.85, 0.87, 0.86, 0.88, 0.85]
model_B_scores = [0.89, 0.91, 0.88, 0.90, 0.92]

t_stat, p_value = ttest_ind(model_A_scores, model_B_scores)
print(f"p-value: {p_value:.4f}")

if p_value < 0.05:
    print("두 모델이 유의하게 다름")
else:
    print("차이가 우연일 수 있음")
```

### ⚠️ p-값의 함정

- p-값이 작다고 **효과가 크다**는 뜻 아님 (그저 "통계적으로 유의")
- 표본 크기가 매우 크면 사소한 차이도 p < 0.05
- p-hacking (이렇게 저렇게 잘라봐서 p < 0.05 나올 때까지) 위험

### ML에서?

- **모델 비교**: A/B 테스트, 두 알고리즘 성능 비교
- **온라인 실험**: 새 모델이 기존 모델보다 진짜 좋은지
- **특성 중요도 검정**

## 5a.5 MLE (Maximum Likelihood Estimation) — 다시 깊게

[5장](05-확률통계-입문.md)에서 잠깐 본 가능도. **모든 ML 모델의 학습이 사실상 MLE.**

### 가능도 (Likelihood)

```
L(θ | 데이터) = ∏ᵢ p(xᵢ | θ)
              ↑
        모든 샘플에 대한 확률의 곱
```

여기서 `θ`는 모델의 파라미터.

### 로그 가능도

곱은 다루기 힘드니 로그 씌우면 합으로 변환.

```
log L(θ | 데이터) = Σᵢ log p(xᵢ | θ)
```

### MLE의 목표

```
θ_MLE = argmax_θ log L(θ | 데이터)
       = argmax_θ Σᵢ log p(xᵢ | θ)
```

**가장 그럴듯한 파라미터 찾기.**

### 예시: 동전의 앞면 확률 추정

10번 던져서 7번 앞면. p (앞면 확률)는?

```
L(p) = C(10,7) × p⁷ × (1-p)³

log L(p) = const + 7 log p + 3 log(1-p)

미분 = 0:
7/p - 3/(1-p) = 0
7(1-p) = 3p
7 = 10p
p = 0.7
```

직관과 같음. **표본 비율이 가장 그럴듯한 p.**

### ML에서 MLE의 등장

#### 선형 회귀 = 가우시안 가정의 MLE

`y = wx + b + ε`, `ε ~ N(0, σ²)` 가정 하의 MLE → MSE 최소화.

#### 로지스틱 회귀 = 베르누이의 MLE

`y ~ Bernoulli(σ(wx+b))` → 이진 cross-entropy.

#### 분류 모델 일반 = 다항분포의 MLE

→ 일반 cross-entropy.

손실 함수의 통계적 기초가 다 MLE예요.

## 5a.6 정보 이론 (Information Theory) — ML의 비밀 무기

샤논(Shannon)이 1948년에 만든 분야. **ML, 특히 분류 모델의 손실 함수**를 진짜로 이해하려면 필수.

### 자기 정보 (Self-Information)

> "어떤 사건이 일어났을 때 우리가 얻는 정보의 양."

```
I(x) = -log p(x)
```

직관:
- 자주 일어나는 사건 (`p` 큼) → 정보 적음 (`I` 작음)
- 드문 사건 (`p` 작음) → 정보 많음 (`I` 큼)

### 예시

- "오늘 해가 떴다" → 거의 매일 일어남 → 정보 적음
- "오늘 일식이 있다" → 드물어서 → 정보 많음

수학적으로:
```
p(해 떴음) = 0.99 → I = -log 0.99 ≈ 0.01
p(일식)   = 0.001 → I = -log 0.001 ≈ 6.9
```

---

### 엔트로피 (Entropy)

> "확률 변수가 가진 **평균 정보량** = 불확실성의 측정."

```
H(X) = -Σᵢ p(xᵢ) × log p(xᵢ)
     = E[-log p(X)]
```

### 직관

- **균일하면 엔트로피 큼** (모르겠음, 무엇이 나올지)
- **편향되면 엔트로피 작음** (확실함)

#### 동전 예시

| 확률 | 엔트로피 |
|------|---------|
| (0.5, 0.5) — 공정한 동전 | 1 bit (가장 불확실) |
| (0.9, 0.1) — 편향된 동전 | 0.47 bit |
| (1.0, 0.0) — 항상 앞면 | 0 bit (확실) |

`log` 의 밑이 2면 단위가 bit, 밑이 e면 nat.

### ML에서?

#### 결정 트리의 분할 기준

엔트로피가 가장 많이 줄어드는 특성으로 분할 → 정보 이득(Information Gain).

#### 모델 평가

`예측의 엔트로피` 가 작으면 → 모델이 확신 있는 답.

---

### Cross-Entropy — 다시 깊게

> "**진짜 분포 p**와 **예측 분포 q**의 차이."

```
H(p, q) = -Σᵢ p(xᵢ) × log q(xᵢ)
```

### 직관

```
p = 진짜 라벨 (one-hot)
q = 모델의 예측 확률

CE = -Σ pᵢ log qᵢ
   = -log q_정답  (one-hot이므로)
```

정답에 해당하는 모델 예측 확률이 1에 가까우면 CE 작음 (좋음).
0에 가까우면 CE는 무한대로 큼 (나쁨).

### 왜 CE가 분류의 손실 함수?

수학적으로:
```
CE 최소화 ⇔ MLE
```

베르누이/다항분포 가정 하에서 MLE를 풀면 자연스럽게 cross-entropy가 나와요. 그래서 표준이 된 거예요.

---

### KL Divergence — 두 분포의 거리

> "분포 p를 분포 q로 근사할 때 잃는 정보량."

```
D_KL(p || q) = Σᵢ p(xᵢ) × log(p(xᵢ) / q(xᵢ))
             = H(p, q) - H(p)
             = CrossEntropy - Entropy
```

### 성질

```
D_KL(p || q) ≥ 0       (항상 0 이상)
D_KL(p || q) = 0       (p = q 일 때만)
D_KL(p || q) ≠ D_KL(q || p)   (대칭 아님!)
```

거리 같은 건데 대칭은 아니에요. 그래서 정확히 거리는 아님.

### CE와의 관계

```
CrossEntropy(p, q) = Entropy(p) + KL(p || q)
                     ↑              ↑
                  진짜 분포의      예측이 진짜에서
                  본질적 정보      얼마나 떨어졌나
```

`Entropy(p)`는 모델이 어쩔 수 없는 부분 (진짜 데이터의 본질). `KL(p||q)`가 모델이 줄여야 하는 부분.

**그래서 CE 최소화 = KL 최소화.**

### ML에서 KL이 어디?

#### 1. VAE (Variational Autoencoder)

잠재 분포가 표준정규에 가까워지도록 KL 페널티.

#### 2. Knowledge Distillation

큰 모델(teacher)의 출력 분포를 작은 모델(student)이 모방. KL로 거리 측정.

#### 3. Reinforcement Learning

PPO 등의 알고리즘에서 정책 변화 제한.

#### 4. 베이지안 ML

posterior 근사할 때.

---

### 상호 정보 (Mutual Information)

두 변수가 **얼마나 의존적**인지.

```
I(X; Y) = D_KL(p(X,Y) || p(X)p(Y))
```

`X, Y` 가 독립이면 `I = 0`. 의존적일수록 큼.

### ML에서?

- **특성 선택**: 타겟과 상호 정보 큰 특성 고르기
- **자기 지도 학습**: 두 view가 같은 의미를 갖도록

## 5a.7 사전 분포와 사후 분포 (Bayesian)

[5장](05-확률통계-입문.md)의 베이즈 정리 다시.

```
P(θ | 데이터) = P(데이터 | θ) × P(θ) / P(데이터)
       ↑              ↑              ↑
    사후분포       가능도        사전분포
   (Posterior)   (Likelihood)    (Prior)
```

### 의미

- **Prior**: 데이터 보기 전 우리의 믿음
- **Likelihood**: 가설(θ)이 사실일 때 이 데이터가 나올 확률
- **Posterior**: 데이터 본 후 갱신된 믿음

### MAP (Maximum A Posteriori)

MLE는 likelihood 최대화. **MAP는 posterior 최대화.**

```
θ_MAP = argmax_θ P(θ | 데이터)
      = argmax_θ [log P(데이터 | θ) + log P(θ)]
                         ↑                  ↑
                       MLE 항목        정규화 항목
```

### MAP = MLE + 정규화

`P(θ)` 가 가우시안이면 → L2 정규화.
`P(θ)` 가 라플라스면 → L1 정규화.

이렇게 보면 정규화도 베이지안 관점에서 **사전 믿음의 표현**이에요.

### ML 식 다시 읽기

```
CE = -Σᵢ yᵢ log p̂ᵢ
```
"교차엔트로피 = 마이너스 (모든 i 에 대해 정답 라벨 × 예측 확률 로그)의 합."

```
KL(p||q) = Σ p log(p/q)
```
"p와 q의 KL = 모든 (p × log(p/q))의 합."

```
H(p) = -Σ p log p
```
"분포 p의 엔트로피 = 마이너스 (p × log p)의 합."

```
θ_MAP = argmax [log P(D|θ) + log P(θ)]
```
"MAP 추정값 = (가능도의 로그 + 사전분포의 로그)를 최대화하는 θ."

다 읽혀요. 진짜로요.

## 자가 진단

1. 표본의 평균은 어떤 분포에 가까워지나요? (정리 이름과 분포)
2. 95% 신뢰 구간 공식은?
3. p-값이 0.03이면 어떻게 해석?
4. `H((0.5, 0.5))` 와 `H((0.9, 0.1))` 중 어느 게 큼?
5. KL divergence 가 0이 되는 경우는?

<details>
<summary>정답</summary>

1. **중심극한정리**, **정규분포** (X̄ ~ N(μ, σ²/n))
2. `X̄ ± 1.96 × (σ/√n)` (또는 SE)
3. `0.03 < 0.05` 이므로 귀무가설 기각, "유의하다"
4. `(0.5, 0.5)` 가 큼 (1 bit > 0.47 bit). 균일할수록 엔트로피 큼.
5. `p = q` 일 때만. 두 분포가 완전히 같으면 KL = 0.
